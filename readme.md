# FUCKBR

this is the decryption tools for BLACK RUSSIA!

---

## requirements
 
- python 3.9+
- all dependencies listed in `requirements.txt`

---

## how to use

for decrypt (`mod`, `btx`, `cls`) files, just use:

```bash
py file_type.py -i C:\input -o C:\output
```

for decrypt `bpc`, use:

```bash
    py bpc.py -i C:\input
    ```
